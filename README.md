# RubyGo

Syntax sugars (`go`, `chan`, `select`) around Ruby threads and queues, emulating Golang Goroutines.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'ruby_go'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install ruby_go
